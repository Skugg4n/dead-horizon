## Fäll-progression per natt

```
Natt 1-2: Nail Boards, Trip Wires, Glass -- primitiva, lätt
Natt 3:   Hittat första blueprinten → Blade Spinner/Fire Pit
Natt 4:   Kombinationer, kill corridors tar form
Natt 5:   BOSS → behöver starka fällor, Meat Grinder etc.
Natt 6+:  Nya zoner → nya blueprints → galnare maskiner
```
